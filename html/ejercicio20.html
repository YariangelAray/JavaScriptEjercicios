Ejercicio 20

Escriba un algoritmo que permita obtener las raíces reales de la ecuación de segundo grado: ax2 + bx + c, siendo X un valor constante.

Pruebas a = 2, b = 5, c = -3 => Tiene raíces reales diferentes. a = 1, b = -6, c = 9 => Tiene raíces reales iguales. a = 1, b = 2, c = 5 => No tiene raíces reales.
